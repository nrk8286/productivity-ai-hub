### On-Page SEO

#### Title Tags
Format: `[Primary Keyword] | [Secondary Keyword] | Productivity AI Hub`
- Max 60 characters
- Include primary keyword
- Make it compelling

Example: "Best AI Automation Tools for Remote Teams 2025 | Complete Guide"

#### Meta Descriptions
- 150-160 characters
- Include primary keyword
- Include call-to-action
- Make it engaging

Example: "Discover the top AI automation tools transforming remote work. Boost productivity by 10x with our expert-tested recommendations. Read now →"

#### Headers (H1-H6)
- One H1 per page (include primary keyword)
- Multiple H2s for main sections
- H3s for subsections
- Use keyword variations naturally

#### URL Structure
```
/blog/[slug]
/blog/[category]/[slug]
/tools/[tool-name]
/templates/[template-name]
```

Keep URLs:
- Short and descriptive
- Include target keyword
- Use hyphens, not underscores
- Lowercase only

#### Internal Linking
- Link to 3-5 related articles
- Use descriptive anchor text
- Create topic clusters
- Update old posts with new links

### Content Creation Process

#### 1. Research Phase
- Analyze top-ranking content
- Identify content gaps
- Check competitor keywords
- Review user questions (Reddit, Quora, forums)

#### 2. Outline Creation
- Hook/introduction
- Problem statement
- Solution overview
- Detailed sections
- Examples and case studies
- Actionable takeaways
- Call-to-action

#### 3. Writing Guidelines
- **Length:** 1,500-2,500 words for blog posts
- **Reading Level:** 8th-9th grade (Hemingway score)
- **Tone:** Professional but conversational
- **Format:**
  - Short paragraphs (2-4 sentences)
  - Bullet points and numbered lists
  - Subheadings every 300 words
  - Bold key points
  - Include images/screenshots

#### 4. Optimization
- Primary keyword in:
  - Title
  - First paragraph
  - One H2
  - URL
  - Meta description
  - Image alt text
- LSI keywords naturally throughout
- Internal links (3-5)
- External links to authoritative sources (2-3)

#### 5. Visual Elements
- Featured image (1200x630px)
- Screenshots where relevant
- Infographics for processes
- Tables for comparisons
- Charts for data
